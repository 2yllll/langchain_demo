# GPT & Langchain 例子

## 1. [准备工作](src/share/00_setup.md)

+ 要 代理，搭梯子
+ 要 OpenAI 账号
+ 要 信用卡绑定 API（注：此处不需要 Plus账号，信用卡是为了根据token扣api费用的）
+ 申请一个 API Key：
    - 将 OpenAI 的 API-KEY 填到 环境变量 `OPENAI_API_KEY` 里

## 2. 运行环境

+ `python 3。10+` 
+ 编辑器：`vscode`
